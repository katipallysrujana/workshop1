Why we use name="viewport" in metatag

we use name="viweport" in the meta tag to control the layout and scaling of a webpage on diferent devices,especially on mobile devices. it provides instructions to browser on how to display the content and set the initial zoom level of the web page

Explore charset,property,http attributes in meta tag

Charset:It ensures that the browser interprets the text correctly by defining the caracter set used in the document
property:It defines specific properties like the title,decription,and image of a webpage for social media platforms.It helps platforms display shared content effectively.
http-equiv:It is used to define a specific HTTP header foe the information containes within the meta tag.This attribute allows you to simulate an HTTP response header within the HTML document.
